Forest area and forest area changes

These menu layers will show area changes related to forest land: a) forest land remaining forest land (such as degradation, regrowth and conservation); b) forest land converted to non-forest land (deforestation); c) non-forest land converted to forest land (afforestation / reforestation); and d) biomass burning. Currently the menu option shows historical deforestation only. When fully active, this layer will also provide annual changes for a), b), c) and d).
